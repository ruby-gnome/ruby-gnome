# Ruby/Clutter

Ruby/Clutter is a Ruby binding of Clutter.

## Requirements

* [Ruby/GLib2](http://www.ruby-lang.org/)
* [Clutter](http://blogs.gnome.org/clutter/)

## Install

  gem install clutter
