# Ruby/Clutter

Ruby/Clutter is a Ruby binding of Clutter.

## Requirements

* Ruby/GObjectIntrospection in
  [Ruby-GNOME2](https://ruby-gnome.github.io/)
* [Clutter](http://blogs.gnome.org/clutter/)

## Install

    gem install clutter
