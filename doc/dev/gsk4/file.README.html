Ruby/GSK4

Ruby/GSK4 is a Ruby binding of GSK 4.
https://docs.gtk.org/gsk4/

Requirements

  • Ruby/GLib2, Ruby/GObjectIntrospection, Ruby/Pango and Ruby/GdkPixbuf2,
    Ruby/GDK4 and Ruby/Graphene in Ruby-GNOME
  • rcairo
  • GTK 4 or later

Install

gem install gsk4
