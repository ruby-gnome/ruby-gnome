Ruby/GTK3

Ruby/GTK3 is a Ruby binding of GTK+ 3.

Requirements

  • Ruby/GLib2, Ruby/ATK, Ruby/Pango, Ruby/GdkPixbuf2, Ruby/GIO2,
    Ruby/GDK3, Ruby/GObjectIntrospection and Ruby/GTK3 in
    Ruby-GNOME
  • rcairo
  • GTK+ 3.4.2 or later

Install

gem install gtk3

Tutorials, examples:

  • sample/misc: a lot of simple examples.
  • sample/gtk-demo: the ruby version of gtk-demo.
  • sample/tutorial: the ruby version of the offical Gtk3 tutorial "getting started".
  • a well written tutorial on how to create a todo application
