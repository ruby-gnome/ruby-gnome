Ruby/GDK3

Ruby/GDK3 is a Ruby binding of GDK 3.

Requirements

  • Ruby/GLib2, Ruby/ATK, Ruby/Pango and Ruby/GdkPixbuf2 in
    Ruby-GNOME2
  • rcairo
  • GTK+ 3.4.2 or later

Install

gem install gdk3
