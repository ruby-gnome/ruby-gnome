Ruby/WebKit2GTK

Ruby/WebKit2GTK is a Ruby binding of WebKit2GTK.

Requirements

  • Ruby/GObjectIntrospection and Ruby/GTK3 in
    Ruby-GNOME
  • WebKit2GTK 2.32.0 or later.

Install

gem install webkit2-gtk
