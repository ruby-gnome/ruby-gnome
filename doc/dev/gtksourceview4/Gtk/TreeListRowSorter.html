#initialize(sorter) ⇒ Gtk::TreeListRowSorter

Create a special-purpose sorter that applies the sorting of sorter to the levels of a GtkTreeListModel.

Note that this sorter relies on [propertyGtk.TreeListModel:passthrough] being false as it can only sort [classGtk.TreeListRow]s.
