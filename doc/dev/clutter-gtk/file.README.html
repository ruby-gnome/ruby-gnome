Ruby/ClutterGTK

Ruby/ClutterGTK is a Ruby binding of Clutter-GTK.

Requirements

  • Ruby/Clutter and Ruby/GTK3 in Ruby-GNOME2
  • Clutter-GTK

Install

gem install clutter-gtk
