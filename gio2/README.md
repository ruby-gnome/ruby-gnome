# Ruby/GIO2

Ruby/GIO2 is a Ruby binding of gio-2.0.x.

## Requirements

* Ruby/GObjectIntrospection in
  [Ruby-GNOME2](https://ruby-gnome.github.io/)
* [GIO](http://library.gnome.org/devel/gio/stable)

## Install

    gem install gio2
