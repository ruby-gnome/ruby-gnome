Ruby/VLC
========

Ruby/VLC is a Ruby binding of libVLC for Ruby/GTK.

Requirements
------------

  Ruby >= 1.9.x: http://www.ruby-lang.org/
  libVLC >= 2.0: http://www.videolan.org/

Install (RubyGems)
------------------

  % sudo gem install gvlc

Install (traditional)
---------------------

Install ruby-1.9.x or later and libVLC >= 2.0.

  % ruby extconf.rb
  % make
  % sudo make install
