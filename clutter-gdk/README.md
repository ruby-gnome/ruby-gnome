# Ruby/ClutterGDK

Ruby/ClutterGDK is a Ruby binding of GDK specific API of Clutter.

## Requirements

* Ruby/Clutter and Ruby/GTK3 in
  [Ruby-GNOME2](https://ruby-gnome.github.io/)
* [Clutter-gdk](http://blogs.gnome.org/clutter/)

## Install

    gem install clutter-gdk
