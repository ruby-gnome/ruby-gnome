# Ruby/GStreamer

Ruby/GStreamer is a Ruby binding for GStreamer.

## Requirements

* Ruby/GLib2 and Ruby/GObjectIntrospection in
  [Ruby-GNOME2](http://ruby-gnome2.sourceforge.jp/)
* [GStreamer](http://gstreamer.freedesktop.org/)

## Install

    gem install gstreamer
