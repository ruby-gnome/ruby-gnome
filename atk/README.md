# Ruby/ATK

Ruby/ATK is a Ruby binding of ATK-1.12.x or later based on GObject-Introspection.

## Requirements

* [Ruby/GObject-Introspection](https://github.com/ruby-gnome/ruby-gnome)
* [Ruby/GLib2](https://github.com/ruby-gnome/ruby-gnome)

## Install

    gem install atk
