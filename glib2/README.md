# Ruby/GLib2

Ruby/GLib2 is a Ruby binding of GLib 2.48 or later.

## Requirements

  * Ruby >= 2.5:   https://www.ruby-lang.org/
  * pkg-config.rb: https://github.com/ruby-gnome/pkg-config
  * GLib >= 2.48:  https://www.gtk.org/

## Install

```bash
gem install glib2
```
