# Ruby/GdkPixbuf2

Ruby/GdkPixbuf2 is a Ruby binding of GdkPixbuf-2.x.

## Requirements

* Ruby/GIO2
* [GdkPixbuf](https://gitlab.gnome.org/GNOME/gdk-pixbuf)

## Install

    gem install gdk_pixbuf2
