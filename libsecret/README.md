# Ruby/libsecret

Ruby/libsecret is a Ruby binding of libsecret.

## Requirements

* Ruby/GObjectIntrospection in
  [Ruby-GNOME2](https://ruby-gnome2.osdn.jp/)
* [libsecret](https://wiki.gnome.org/Projects/Libsecret)

## Install

    gem install libsecret
