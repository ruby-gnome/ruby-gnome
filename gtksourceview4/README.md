# Ruby/GtkSourceView4

Ruby/GtkSourceView4 is a Ruby binding of gtksourceview-4.x.

## Requirements

* Ruby/GTK3 in
  [Ruby-GNOME2](https://ruby-gnome2.osdn.jp/)
* [GtkSourceView](http://projects.gnome.org/gtksourceview/) 4.0.0 or later

## Install

    gem install gtksourceview4
