# Ruby/GtkSourceView3

Ruby/GtkSourceView3 is a Ruby binding of gtksourceview-3.x.

## Requirements

* Ruby/GTK3 in
  [Ruby-GNOME2](https://ruby-gnome.github.io/)
* [GtkSourceView](http://projects.gnome.org/gtksourceview/) 3.4.2 or later

## Install

    gem install gtksourceview3
