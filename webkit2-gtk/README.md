# Ruby/WebKit2GTK

Ruby/WebKit2GTK is a Ruby binding of WebKit2GTK+.

## Requirements

* Ruby/GObjectIntrospection and Ruby/GTK3 in
  [Ruby-GNOME](https://ruby-gnome2.osdn.jp/)
* [WebKit2GTK+](http://webkitgtk.org/) 2.32.0 or later.

## Install

    gem install webkit2-gtk
