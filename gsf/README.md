# Ruby/GSF

Ruby/GSF is a Ruby binding of GSF which is needed by GOffice.

## Requirements

* Ruby/GIO2 in
  [Ruby-GNOME2](https://ruby-gnome2.osdn.jp/)
* [GSF](https://developer.gnome.org/gsf/)

## Install

    gem install gsf
