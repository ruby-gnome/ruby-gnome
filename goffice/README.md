# Ruby/GOffice

Ruby/GOffice is a Ruby binding of GOffice.

## Requirements

* Ruby/GTK3 and Ruby/GSF in
  [Ruby-GNOME2](https://ruby-gnome.github.io/)
* [GOffice](https://developer.gnome.org/goffice/)

## Install

    gem install goffice
