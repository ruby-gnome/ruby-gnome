# Ruby/VTE3

Ruby/VTE3 is a Ruby binding of VTE for use with GTK3.

## Requirements

* Ruby/GTK3 in
  [Ruby-GNOME2](http://ruby-gnome2.sourceforge.jp/)
* [VTE](https://live.gnome.org/Terminal/VTE)

## Install

    gem install vte3
