# Ruby/CairoGObject

Ruby/CairoGObject is a Ruby binding of cairo-gobject.

## Requirements

* Ruby/GLib2 in
  [Ruby-GNOME2](https://ruby-gnome.github.io/)
* [rcairo](https://github.com/rcairo/rcairo)
* [cairo](http://cairographics.org/) GObject binding

## Install

    gem install cairo-gobject
