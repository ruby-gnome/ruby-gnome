# Ruby/Gnumeric

Ruby/Gnumeric is a Ruby binding of Gnumeric.

## Requirements

* Ruby/GOffice in
  [Ruby-GNOME2](https://ruby-gnome.github.io/)
* [Gnumeric](http://www.gnumeric.org/)

## Install

    gem install gnumeric
