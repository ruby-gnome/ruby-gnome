# Ruby/GEGL

Ruby/GEGL is a Ruby binding of GEGL.

## Requirements

* Ruby/GObjectIntrospection in
  [Ruby-GNOME2](https://ruby-gnome.github.io/)
* [GEGL](https://gegl.org/)

## Install

    gem install gegl
