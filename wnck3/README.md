# Ruby/WNCK3

Ruby/WNCK3 is a Ruby binding of libwnck 3.

## Requirements

* Ruby/GObjectIntrospection in
  [Ruby-GNOME2](https://ruby-gnome2.osdn.jp/)
* [libwnck](https://developer.gnome.org/libwnck/stable/)

## Install

    gem install wnck3
